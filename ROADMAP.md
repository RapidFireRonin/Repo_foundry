# Roadmap

## Phase 1

- Repo scaffold, schemas, reconcile planning, tests, dashboard, docs, and workflows.
- Completion stewardship contract: PR/check snapshots, policy classification, cycle logs, and dashboard-readable status.

## Phase 2

- GitHub App integration, live PR creation, branch protection checks, artifact sync, and report-first hosted workflows.
- Local runner bridge for policy-bounded repository maintenance when hosted workflow credentials are intentionally limited.

## Phase 3

- Scheduled multi-agent cycles, direction queues, cross-repo desired-state reconciliation, and autonomous completion loops that can safely land low-risk work while preserving human directability.
